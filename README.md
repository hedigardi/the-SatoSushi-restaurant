## Installation
1. Clone the SatoSushi-restaurant repository:
   ```bash
   git clone https://github.com/hedigardi/the-SatoSushi-restaurant.git
    ```
2. Install the dependencies:
   ```bash
   npm install
    ```
3. Start the application:
   ```bash
   npm run dev
    ```
